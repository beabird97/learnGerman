Import Verbs (CSV)

CSV Format: infinitive,english,ich,du,er_sie_es,wir,ihr,sie_Sie,level
Example: 
laufen,run,laufe,läufst,läuft,laufen,lauft,laufen,A1
sprechen,speak,spreche,sprichst,spricht,sprechen,sprecht,sprechen,A2
denken,think,denke,denkst,denkt,denken,denkt,denken,B1
Valid levels: A1, A2, B1, B2, C1 (defaults to A1 if not specified)
Note: Duplicates (same infinitive) will be skipped.
